# Проектный документ

## 1. Цель и показатели
**Задача:** предсказать зрительский рейтинг (IMDb / MAL) до выхода фильма или аниме, используя только предрелизные характеристики.  
**Метрики:** MAE, RMSE, R². Основная метрика – R².  
**Критерий успеха:** R² > 0.4 на объединённой выборке, сравнение важности признаков между кино и аниме.

## 2. Данные
- **Фильмы:** "The Ultimate Film Statistics Dataset", 4 380 записей (1915–2023). Признаки: название, год, жанры, длительность, режиссёр, рейтинг, бюджет.
- **Аниме:** "Anime Dataset 2024", 19 311 записей. Признаки: название, жанры, год, эпизоды, рейтинг, синопсис.
- **Объединение:** После обработки пропусков и приведения к общему виду получено 23 691 записей. Объединение выполнено по одинаковым колонкам, разнородные форматы (жанры строкой / списком) нормализованы.

## 3. Проект для проверки
- Отложенная выборка: train/test 80/20, random_state=42.
- Целевая переменная для общей модели нормализована (z-score) для устранения разницы шкал IMDb и MAL.
- Строгое разделение train/test до любых вычислений статистик (target encoding) – утечка данных исключена.

## 4. Подход
- **Признаки:** год, общая продолжительность (total_runtime), длина синопсиса, кол-во слов, логарифм бюджета, флаг современности, репутация режиссёра (средний рейтинг по тренировочным данным), флаг крутого режиссёра, 20 топ-жанров, бинарный признак is_anime.
- **Baseline:** Ridge-регрессия.
- **Эксперименты:** Random Forest, Gradient Boosting.

## 5. Обучение
Лучшие гиперпараметры: Gradient Boosting (n_estimators=100, max_depth=5, learning_rate=0.1).  
Анализ ошибок: MAE ~0.57 (нормализованная шкала), ошибка в пределах 0.5–0.7 балла исходной шкалы.

## 6. Проверка
- **SHAP-анализ:** главный универсальный фактор – `director_mean_rating`. Для аниме выше важность текстовых признаков и числа эпизодов. Флаг `is_anime` влияет через взаимодействия.
- Остатки распределены нормально, явных выбросов нет.

## 7. Выбор модели
- **Итоговая модель:** Gradient Boosting (общая выборка), R²=0.44.
- **Риски:**
  - Разные шкалы оценок IMDb и MAL – частично скомпенсировано нормализацией.
  - Отсутствие бюджета у аниме – модель учится компенсировать через другие признаки.
  - Неполнота данных о создателях для аниме.
- **Ограничения:** предсказание ограничено историческими данными, не учитывает культурный контекст.

## 8. Развертывание
Модель может быть встроена в Telegram-бота. Бот принимает тип (фильм/аниме), жанры, год, длительность и возвращает прогноз рейтинга. Переобучение – раз в год при поступлении новых данных.  
Инфраструктура: Python-скрипт на сервере.
