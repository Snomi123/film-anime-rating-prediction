# ML-анализ факторов успеха в кино и аниме

## Цель проекта
Предсказать зрительский рейтинг (IMDb для фильмов, MAL для аниме) на основе предрелизных характеристик и выяснить, различаются ли факторы успеха в двух индустриях.

## Данные
- **Фильмы** (`film_statistics.csv`) – ~4300 фильмов (1915–2023) с признаками: жанры, режиссёр, бюджет, длительность, рейтинг.
- **Аниме** (`animes.csv`) – ~19 300 аниме с признаками: жанры, эпизоды, синопсис, рейтинг.
- Данные объединены в общий DataFrame (~23 000 записей) после тщательной предобработки.

## Используемые методы
Только классические модели:
- Ridge-регрессия (baseline)
- Random Forest
- Gradient Boosting

Предобработка:
- Объединение двух разнородных источников
- Извлечение года и длительности из строк
- Логарифмирование бюджета
- Создание признаков: репутация режиссёра, длина синопсиса, нормализация рейтинга
- One-hot кодирование жанров (MultiLabelBinarizer)

## Результаты
- **Лучшая модель** – Gradient Boosting на общей выборке: R² = 0.44 (нормализованный рейтинг)
- Модель для аниме показала R² = 0.50 (Gradient Boosting)
- Модель для фильмов – R² = 0.22 (честная, без утечек)

**Ключевые факторы успеха (SHAP):**
1. Репутация режиссёра (`director_mean_rating`)
2. Длина синопсиса (`synopsis_len`)
3. Общая продолжительность (`total_runtime`)
4. Год выпуска
5. Жанры: Drama, Shounen

## Источники данных
- Фильмы: [The Ultimate Film Statistics Dataset](https://www.kaggle.com/datasets/alessandrolobello/the-ultimate-film-statistics-dataset-for-ml?resource=download)
- Аниме: [Anime Dataset 2024](https://www.kaggle.com/datasets/arnavvvvv/anime-dataset/data)

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Скачать датасеты (ссылки в отчёте) и поместить в папку `data/`
3. Запустить Jupyter Notebook и последовательно выполнить ячейки `main.ipynb`
